Diminished Triads

Learn how diminished triads are built, then practice telling them apart from major and minor triads.

A diminished triad is built with a minor third and a diminished fifth. That means the top note is one half step lower than it would be in a minor triad.

  1. Start on the root
  2. Add a minor third: 3 half steps
  3. Add a diminished fifth: 6 half steps

C diminished

To build C diminished, start on C. Count 3 half steps to E♭, then 6 half steps to G♭. The triad is C–E♭–G♭.

C–E♭–G♭
The fifth is lowered compared to C minor.

D diminished

To build D diminished, start on D. Count 3 half steps to F, then 6 half steps to A♭. The triad is D–F–A♭.

D–F–A♭
Another example of a diminished triad.

General rule

  1. Major = 4, 7
  2. Minor = 3, 7
  3. Diminished = 3, 6
